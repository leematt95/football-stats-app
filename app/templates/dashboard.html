API Endpoints

Available Endpoints:
  • GET /api/players/ - All players
  • GET /api/players/?name=search - Search players
  • GET /api/players/{id} - Specific player
  • GET /api/players/?limit=N - Limited results
Example Usage:
curl "http://localhost:5000/api/players/?name=haaland&limit=5"
